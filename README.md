## Rule
- 스터디는 매주 일요일 10시에 진행
- 월~토까지 특정 주제를 조사하고 내용을 정리하여 스터디 시간에 발표
- 조사 내용
-- 기본 내용/ 동작원리/ 장단점
-- CS의 실무 사용 / 예시
-- 관련 내용의 실제 구현
- 발표 이후 미흡하거나 질의에 제대로 대답하지 못한 부분을 추가 조사 ~ 일요일 23:59까지
